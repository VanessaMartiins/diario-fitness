🌱 O que é este projeto?

O Diário Fitness é um projeto pessoal criado com o objetivo de acompanhar hábitos relacionados à alimentação e bem-estar de forma simples, visual e acessível.

🎯 Objetivo

A ideia principal é incentivar a constância, o autoconhecimento e o registro consciente de hábitos, sem foco em números extremos ou cobranças excessivas.

🛠️ Tecnologias utilizadas

  • HTML5
  • CSS3
  • JavaScript (vanilla)
  • LocalStorage para persistência de dados
  • GitHub Pages para publicação

⚠️ Aviso importante

Este site não possui finalidade médica ou nutricional. Todas as informações apresentadas são apenas para uso pessoal e educacional. O acompanhamento com profissionais de saúde é essencial.
