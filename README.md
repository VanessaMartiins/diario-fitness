🌱 Diário Fitness

Projeto pessoal de acompanhamento de hábitos saudáveis, desenvolvido com HTML, CSS e JavaScript puro, com foco em organização, constância e experiência do usuário.

⚠️ Aviso: este projeto não possui finalidade médica ou nutricional. O acompanhamento com profissionais de saúde é essencial.

📌 Visão Geral

O Diário Fitness é uma aplicação web simples e intuitiva que permite:

Calcular referências saudáveis (IMC)

Definir metas semanais de acompanhamento

Registrar refeições com foto e comentário

Salvar dados localmente no navegador

Compartilhar o site com amigos

Manter um histórico pessoal de hábitos

O projeto foi criado como parte do portfólio pessoal, unindo desenvolvimento front-end, lógica em JavaScript e preocupação com usabilidade.

✨ Funcionalidades

📊 Cálculo saudável (IMC)
Aceita altura em metros ou centímetros e peso com vírgula ou ponto.

🎯 Meta semanal
Geração de sugestão de acompanhamento com salvamento automático.

🍽️ Registro de refeições

Upload de imagem

Comentário personalizado

Data automática

Histórico persistente

🗑️ Gerenciamento

Exclusão de refeições individuais

📤 Compartilhamento

Compartilhar o site via WhatsApp

Copiar link do site

💾 Persistência de dados

Uso de localStorage (sem backend)

🛠️ Tecnologias Utilizadas

HTML5 — estrutura semântica

CSS3 — layout responsivo e visual minimalista

JavaScript (Vanilla) — lógica, interatividade e persistência

LocalStorage — armazenamento local de dados

GitHub Pages — hospedagem
